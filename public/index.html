Projects

BiocSwirl

Open Source

A series of in-depth and easily updateable swirlify made courses used to teach entire bioinformatics workflows in R/Bioconductor using an interactive and easy to digest format. Winner of Hackseq2019.

BiocTerm

Open Source

A highly configurable, lightweight and feature rich terminal interface with multi-language support for conducting bioinformatics. BiocTerm currently supports R, Bash, Python, and Vim. Winner of Hackseq2019.

behavendoR (early stages)

Open Source

An R package used to introduce statistical workflows and reproducibility into the Watson lab and Behavioural Endocrinology field, has basic features that improve R usability for behavioural endocrinologists.
